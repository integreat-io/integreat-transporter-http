# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.4.x   | :white_check_mark: |
| < 1.4   | :x:                |

## Reporting a Vulnerability

To report a vulnerability, please
[create an issue](https://github.com/integreat-io/integreat-transporter-http/issues)
in this repo.
